Manifest
--------

**Android 9.0.x** (latest is 9.0.0_r3)

1. Initialize the manifest repo		
`repo init -u https://github.com/krexus/manifest.git -b p`

2. Sync		
`repo sync`
 
3. Lunch		
`lunch`

4. Build!		
`mka otapackage`
