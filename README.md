Manifest
--------

**Android 8.1.x** (latest is 8.1.0_r18)

1. Initialize the manifest repo		
`repo init -u https://github.com/krexus/manifest.git -b o+`

2. Sync		
`repo sync`
 
3. Lunch		
`lunch`

4. Build!		
`mka otapackage`
